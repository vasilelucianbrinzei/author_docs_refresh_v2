# Core Workshop Flow

## Introduction

Use this section for the required author path from workshop request through GitHub setup, repository sync, and content development. These labs represent the main sequence most authors should follow.

Estimated Time: 5 minutes

### Objectives

* Understand the main author workflow in the correct order
* Know which labs are required for most workshop authors
* Move from request to authoring without skipping critical setup steps

## Task 1: Follow The Main Author Path

1. Start with WMS submission and workflow so you understand approvals and status changes.
2. Complete GitHub and local tool setup before you begin editing workshop content.
3. Continue through repository sync and Markdown authoring in order before moving on to QA and publishing.

## Included Labs

* Lab 1: Submit new workshop in WMS
* Lab 2: Set up GitHub and install tools
* Lab 3: Stay in sync with GitHub environment
* Lab 5a: Develop Markdown and content
